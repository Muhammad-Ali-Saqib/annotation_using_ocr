# annotation_using_ocr
We can perfrom annotations using OCR.
Annotations are done in YOLO format.

# installation
```
python -m pip install paddlepaddle -i https://pypi.tuna.tsinghua.edu.cn/simple
pip install "paddleocr>=2.0.1"
```
